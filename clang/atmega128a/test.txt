비트 마스킹

비트를 제어할 때 (일단 다른 비트는 고려하지 말것!!)
원하는 위치에 1을 넣고 싶으면
원하는 위치만큼 1을 시프트 시키고 OR 연산을 실행
예)
PORTA의 4번에 1을 넣고 싶으면
PORTA |= (1 << 4);      // 이렇게 함

원하는 위치에 0을 넣고 싶으면
원하는 위치만큼 1을 시프트 시키고  NOT 연산을 걸어서 원하는 위치를 0을 만듦
-> 기존 PORT와 AND 연산을 함
예)
PORTA의 3번에 0을 넣고 싶으면
PORTA &= ~(1 << 3);     // 이렇게 함

원하는 위치에 0인지? 1인지? 확인하는 방법은
원하는 위치까지 1을 시프트 시켜서 포트와 AND 연산을 함!!
예)
PORTA의 2번에 있는 값을 알고 싶으면
PORTA &= (1 << 2);
